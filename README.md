# Basketball Scoreboard

A simple basketball scoreboard I built with plain HTML, CSS, and JavaScript.

**Live demo:** https://sspatel107.github.io/basketball-scoreboard/

## Features

- Track scores for HOME and GUEST with +1, +2, and +3 buttons
- The leading team's score is highlighted
- Per-team foul counters
- Period counter (1–4)
- A 10-minute game clock with start/pause and reset
- Winner announcement when the clock hits 0:00
- A New Game button that resets everything

## Tech

- HTML
- CSS
- JavaScript

## Run locally

Clone the repo and open `index.html` in your browser.
